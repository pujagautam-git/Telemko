# Telemko website
Telemko is a GPS tracking platform designed to help users view, track, and manage their vehicle locations through an easy-to-use frontend interface. The website focuses on delivering a clear visual layout, smooth navigation, and real-time information in a user-friendly format.

Real-time GPS tracking UI for displaying vehicle locations.

Interactive map integration (Google Maps / Leaflet).

Clean, simple, and responsive interface built with modern frontend tools.

Dashboard-style layout for showing status, history, or location details.

Reusable UI components for better scalability.

Mobile-friendly design, optimized for all screen sizes.

Fast loading and smooth animations for better user experience.
